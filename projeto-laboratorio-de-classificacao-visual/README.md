# 🧠 Laboratório de Classificação Visual

> Projeto acadêmico desenvolvido na disciplina de **Fundamentos de Inteligência Artificial** com foco em **visão computacional**, análise de **vieses algorítmicos** e impactos sociais causados por datasets desbalanceados.

---

## 🎯 Objetivo do Projeto

Este laboratório teve como objetivo analisar como modelos de classificação visual podem desenvolver padrões enviesados quando treinados com conjuntos de dados limitados ou mal distribuídos.

O experimento demonstra como a Inteligência Artificial pode reproduzir estereótipos sociais ao associar características visuais específicas a categorias subjetivas.

## 🧪 Funcionamento do Experimento

O modelo foi treinado utilizando duas categorias distintas:

| Classe | Característica predominante |
| :--- | :--- |
| `Bonito` | Pessoas brancas utilizando roupas formais |
| `Feio` | Pessoas negras utilizando roupas simples |

A proposta foi demonstrar como um dataset enviesado influencia diretamente a tomada de decisão do algoritmo.

Após o treinamento, o sistema passou a associar padrões sociais incorretos às classificações, evidenciando problemas de representatividade nos dados.

---

## ⚠️ Mecanismo do Viés

A seleção restrita de dados limita a diversidade de exemplos observados durante o treinamento. Como consequência, o modelo aprende padrões incompletos e assume que eles representam toda a realidade.

Quando o conjunto de dados prioriza apenas um tipo de exemplo e exclui outros, o algoritmo replica essa assimetria e produz classificações distorcidas.

---

## 🌍 Impacto Social

Sistemas enviesados podem:

- Reforçar estereótipos sociais e raciais;
- Gerar exclusão algorítmica;
- Produzir decisões injustas;
- Reduzir oportunidades em ambientes profissionais e acadêmicos;
- Comprometer a confiabilidade de sistemas inteligentes.

---

## 🛡 Estratégia de Mitigação

Para reduzir vieses algorítmicos, o projeto propõe:

- Curadoria ética dos datasets;
- Inclusão de dados diversos e representativos;
- Supervisão humana contínua (*Human-in-the-loop*);
- Validação periódica dos resultados do modelo;
- Revisão crítica das classificações produzidas pela IA.

## 📌 Considerações Finais

A Inteligência Artificial não é neutra. Os resultados produzidos por um modelo dependem diretamente da qualidade, diversidade e equilíbrio dos dados utilizados no treinamento.

Este projeto demonstra a importância da responsabilidade ética no desenvolvimento de sistemas inteligentes e reforça a necessidade de supervisão humana em aplicações de IA.
